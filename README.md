# employee_tracker

## Purpose: 
Using the command line, view and update all personnel info within your company. <br>
Add a new employee, change their role within your organization, and view their role and salary info on demand.

## Tech:

### Languages:
- Javascript
- node.JS

### Dependencies:
- Inquirer
- MySQL
- console.table
